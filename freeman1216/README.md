### Codegolfed blinky for a challenge by Lars Thomsen from stm32world 

Final size : 18b

To build write
```shell
make
```
To flash (make sure openocd is installed) 
```shell
make flash
```

If you are using ST's flashers just flash the binary at 0x08000000
